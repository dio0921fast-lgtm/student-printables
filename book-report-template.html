How to use this book report template

  1. Write the book title, author, genre, student name, and date at the top.
  2. List the main characters and describe where and when the story happens.
  3. Write the main problem or conflict in one short sentence.
  4. Summarize the beginning, middle, and ending in separate boxes.
  5. Add a favorite part, new words, and a rating or recommendation.
  6. Use the completed template as a final report or as notes for a longer written book report.

What to include in a book report

  • Book title and author
  • Genre or type of book
  • Main characters
  • Setting
  • Problem or conflict
  • Beginning, middle, and ending summary
  • Favorite part and why it stood out
  • New vocabulary words
  • Rating, opinion, or recommendation

Book report tips for students

Students should keep the summary short and focus on the most important events. A good book report does not need to retell every chapter. It should show that the student understands the characters, setting, problem, major events, and personal response to the book.

For younger students, this template can be filled in with short phrases. Older students can use each box as a planning space before writing a paragraph or essay. Teachers can also pair this book report template with a reading log printable so students track reading progress before writing the final report.
