What this printable is for

This class attendance tracker is useful for students, tutors, teachers, homeschool families, and small study groups that need a simple paper attendance record. It keeps attendance notes visible without using a spreadsheet.

Students can print the worksheet, fill it in by hand, and keep it in a binder, folder, planner, or class notebook for quick reference.

How to use it

  1. Write the student name, class, date, or goal at the top.
  2. Fill in the worksheet sections before starting the study task.
  3. Check off completed items or update notes as work changes.
  4. Print a fresh copy whenever you need a new plan.
