How to use this template

  1. Read the prompt and underline the task words.
  2. Write a working thesis before drafting.
  3. Plan each body paragraph with one clear topic sentence.
  4. Add evidence or examples before writing the full essay.
  5. Use the conclusion box to restate the main point and final thought.

Example use case

A student writing a literature essay can plan three body paragraphs before opening a document. Each row holds one claim, one quote, and a short explanation, which makes the first draft easier to write and revise.

Printing tips

Print this page on US Letter or A4 paper. Choose fit to page in the print window so the worksheet stays inside the margins. Students can keep a copy in a binder, tape it near a desk, or place it in a dry erase sleeve for repeated use. For classroom use, print several copies and let students choose a fresh page when a new week, unit, project, or exam begins.
