How to use this essay planner

  1. Write the prompt in your own words.
  2. Draft a clear thesis statement.
  3. Plan each paragraph with one main idea.
  4. Add evidence before you start writing.

Who this essay planner template is for

This essay planner template is for students who need help organizing prompts, thesis statements, body paragraphs, evidence, and conclusion notes before writing. It is useful for English class, history essays, and college prep writing.

How students can use it

Students can start by copying the prompt, then draft a thesis and outline each paragraph before writing the full essay. Planning first can make the final draft more focused and easier to revise.

Tips for printing and reusing it

Print a copy before each essay assignment and use it during brainstorming. You can save a blank PDF version for repeated use throughout the school year.
