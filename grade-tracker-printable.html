What is included?

  • Class and term fields

How to use this grade tracker

  1. Write the course name and grading period.
  2. Add each graded task after you receive it.
  3. Record points earned and points possible.
  4. Use the notes column for retakes, missing work, or study reminders.

Who this grade tracker printable is for

This grade tracker printable is for students who want to monitor scores, possible points, and notes for quizzes, homework, tests, and projects. It makes grades easier to understand between report cards.

How students can use it

Students can enter each graded task, the score received, possible points, and any notes about retakes or missing work. Seeing grades on paper can help identify which assignments have the biggest effect on the final grade.

Tips for printing and reusing it

Print one tracker for each class or grading period. Keep it updated after each returned assignment so parent meetings, teacher conferences, and study decisions are based on current information.
