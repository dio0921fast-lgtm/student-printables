About this printable

A research notes template helps students collect information without losing where it came from. It is useful for essays, presentations, science projects, history reports, and college research assignments.

This research notes template keeps sources, facts, page numbers, follow-up questions, and possible uses together. Students can print one page per source or one page per topic, then use it when outlining or drafting.

How to use this template

  1. Write the research question before taking notes.
  2. Record source details as soon as you use a source.
  3. Write facts or quotes clearly and briefly.
  4. Add page numbers or URLs so you can find the source again.
  5. Use the final columns to connect notes to your outline.

Example use case

A student writing about renewable energy can print one page for each article, record the most useful facts, and write how each fact might support a body paragraph. The notes stay organized before the essay draft begins.

Printing tips

Print this page on US Letter or A4 paper. Choose fit to page in the print window so the worksheet stays inside the margins. Students can keep a copy in a binder, tape it near a desk, or place it in a dry erase sleeve for repeated use. For classroom use, print several copies and let students choose a fresh page when a new week, unit, project, or exam begins.
